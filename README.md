## 项目介绍

- 本项目基于 webpack5 搭建 vue3 + ts 项目。

- 你可以用此项目搭建属于你的项目！！

- 本框架所使用的是 以下最新版本， vue3 typescript4

- "vue-class-component": "^8.0.0-rc.1",

- "vue-loader": "^17.0.0",

- "vue-property-decorator": "^10.0.0-rc.3",

# 安装依赖
npm install

# 进入开发
npm start

# 打包
npm run build
